# Premier Golf

## Setup

### Google Cloud Platform

- Create project
- Create Google Compute Instance
    - EC2
    - Ubuntu
    - 30 gbs storage
- Create SSH key
    - Setup ssh alias
- Install 
    - Docker
    - Docker compose
    - Terraform
- Create Service Account
    - download json file
    - activate google cloud credentials

* Note if restarting instance, need to update ssh config with new external ip address
